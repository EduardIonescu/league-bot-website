What does the bot EVEN do?

Manage Users

Add and remove users to be later used in the checking and betting process

Check Active Matches

Check to see if users are ingame. You can then use the buttons from this message to start betting instances.

Place Bets On Matches

You can start a betting instance on a user's match and bet on it against your friends in the discord server.

Handle Results

The bot automatically handles the results of a match when it ends. Everyone can see who the biggest winners and losers are.
